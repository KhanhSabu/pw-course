Day 1:
Ưu điểm playwright:
- Crossbrowser: firefox, chrome, safari, webkit
- Crossplatform: linux, mac os , window
- Auto-wait: 5s
- Web-first assertion: cú pháp viết nhanh, gọn gàng hơn và cơ chế tự động chờ
- Tracing: Report, hiển thị từng step, pass fail, chạy trên trình duyệt nào, số lượng request
- Code-gen: click UI tự động gen code
- Chạy rất nhanh: so sánh vs selenium

import {test} from '@playwright/test';
Basic syntax
test: test case 
test.step: các bước trong test case
test.describe : to nhất, gom các test case thành test suite

--Checkbox, Radio

const isChecked = await page.locator("").isChecked();

--Selection (Thẻ select có <option> ở phía trong)
page.locator("@id='abc'").selectOption("Vietnam")
page.locator("@id='abc'").selectOption(["Music","Art"])

--Date picker, slider, color picker
page.locator("@id='dob'").fill("01/09/1997") //calendar
page.locator("@id='rating'").fill("3") //rating dạng kéo
page.locator("@id='color'").fill("#e66456") //rating dạng kéo

--File, hover
page.locator("file element").setInputFiles("tests/abc/file.txt") -> nên để file sẵn trước vào testdata
page.locator("hover element").hover() // cách hover để hiển thị tooltip

--Confirmation dialog
page.on('dialog', async dialog => dialog.accept());
page.on('dialog', async dialog => dialog.dismiss());
page.on('dialog', async dialog => console.log(dialog.message()));
